Gestion Sencilla de una Flota de Vehiculos.

El proyecto es una implementacion en C++ que simula un sistema de gestion para un estacionamiento, aplicando conceptos fundamentales de la Programacion Orientada a Objetos (POO), como lo es la herencia, polimorfismo y la agregacion.

Permite crear diferentes tipos de automoviles (coches, camiones y motocicletas) y agruparlos bajo una entidad de estacionamiento para la realizacion comunes como agregar, mostrar, encender y apagar todos los vehiculos.
