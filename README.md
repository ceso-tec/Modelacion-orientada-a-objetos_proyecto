Gestion Sencilla de una Flota de Vehiculos.

El proyecto simula la gestion basica de una flota de vehiculos utilizando la Programaci√≥n Orientada a Objetos en C++.

El objetivo es modelar tres tipos de objetos: Coche, Camion y Motocicleta.  Con caracteristicas y comportamientos unicos.  Este primer avance se enfoca en definir las clases y sus metodos correspondientes.

El programa realiza las siguientes acciones:
1. Creacion de objetos: Coche, Camion y Motocicleta.
2. Inicializacion: Asigna valores iniciales a los atributos privados usando el constructor.
3. Ejecucion de metodos: Llama a los metodos propios de cada clase para simular acciones.
4. Encapsulamiento: Demuestra el acceso controlado a los atributos a traves del metodo getter.
